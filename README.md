Allegro project that prints a bitmap to the screen. The bitmap is 64 x 64 and uses 4 primitives and 4 colors. It's meant to look like a spaceship that points 4 ways!
An ellipse for the ship body, a circle for the ship head (or the compartment a captain sits), a rectangle to represent the person sitting inside.
The blue triangle points the direction of the ship. 
Just downloading the code as a zip and extracting it should work as long as Allegro is installed, needs Dialog and Primitive addons.
